# Project instructions

Build a simple, kid-friendly web-based math game for grade 2 students.

## Requirements
- Use only HTML, CSS, and vanilla JavaScript
- Keep everything beginner-friendly
- Do not add React, TypeScript, or backend code
- Make the design colorful, simple, and easy for children to use
- Use large buttons and large text
- Focus on grade 2 math skills:
  - addition within 100
  - subtraction within 100
  - simple word-problem style prompts later
- Keep files small and readable
- Add comments in the code
- Do not overcomplicate the project

## First goals
1. Create a home screen
2. Add game modes for addition and subtraction
3. Add score tracking
4. Add sounds only if simple
5. Add encouragement messages for kids
6. Make it work well on laptop and tablet

## Important
Explain changes clearly in commit messages and pull request notes.
